## 示範資料佈置

- **帳戶:**
    - 現金與銀行帳戶與信用卡為台幣
    - 日幣旅費為日幣
    - 美金儲蓄為美元
- **類別:**
    - 支出為餐飲、交通、購物、娛樂、居家、醫療、旅遊
    - 收入為薪資、投資
- **交易:**
    - 二十四筆落在 2026 年七月
    - 金額取日常生活合理值
    - 七月支出合計 14862 台幣
    - 七月收入合計 55200 台幣
- **轉帳:**
    - 銀行帳戶提款至現金
    - 銀行帳戶換匯至日幣旅費
    - 銀行帳戶定存至美金儲蓄
- **匯率:**
    - 日幣、美元、歐元、韓元對台幣雙向各一組
    - 生效日皆早於截圖日

## 重製方式

- 模擬器 app 走主 git main 版 dev build
- Metro 跑在 port 8081
- 佈置資料以 sqlite 直寫模擬器 watermelon.db
- 導航與畫面操控走 Metro CDP 通道
- dev 浮層與錯誤提示以 CDP 關閉後再截圖
- 截圖指令與縮放指令如下

```bash
xcrun simctl io booted screenshot <輸出檔>.png
sips -z 2778 1284 <輸出檔>.png
```
