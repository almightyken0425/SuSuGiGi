# 商業模式與定價策略

## 記帳 App + Firestore, Accounting App

- **LEVEL_0**
    - **功能:**
        - 完整記帳功能, CRUD
        - 外幣帳戶, 含手動匯率管理
        - 本地資料庫, WatermelonDB
        - 基礎報表, 圓餅圖、趨勢圖
        - 定期交易排程
        - 資料匯入與匯出, CSV
        - Widget 小工具
        - 資料蒐集, 系統端內部備份且 user 不取回
          - 分析用途採 opt-out, 依 Root Value 隱私承諾
          - 所有 LEVEL 皆可在 Settings 之下 Privacy 關閉
    - **限制:**
      - 帳戶數上限 3 個
      - 類別數上限 7 個

- **LEVEL_1**
    - **價值主張:**
      - 解除帳戶 / 類別 / 交易的總數上限
    - **功能:**
        - **包含 LEVEL_0 所有功能**
        - **無上限帳戶與類別:**
          - 帳戶數與類別數無上限，交易筆數不受總數上限連帶限制

---

## 邏輯引擎與規則中心, Logic Engine

- **LEVEL_2**
    - **價值主張:**
      - 駕馭複雜性，個人財務的邏輯除錯器
    - **功能:**
        - **包含 LEVEL_1 所有功能**
        - **規則中心:**
          - 無限量的自動化規則 Mad Libs UI
        - **Web 控制台:**
          - 大螢幕管理與 JQL 查詢
          - 資料來源為 user 主動上傳的匯出檔，採瀏覽器端儲存
        - **時光機回溯:**
          - 規則修改後的歷史重構 Replay

---

## AI 個人化建議, AI Advisor

- **LEVEL_3**
    - **價值主張:**
      - 專屬財務顧問，預知未來
    - **功能:**
        - **包含 LEVEL_2 所有功能**
        - **AI Agent:**
          - 自然語言邏輯除錯 Chat Debugging
        - **異常偵測:**
          - 財務健康診斷報告內被動標示異常消費
          - 不做主動推播, AI 顧問定位為陪伴輔助
        - **現金流預測:**
          - 未來財務狀況模擬
        - **財務健康:**
          - 診斷報告與建議

---

## 總經分析報表與 API, Macro Data Service

- **LEVEL_B**
    - **客戶:**
      - 市場研究機構、品牌商、金融機構
    - **產品:**
        - **市場情資儀表板:**
          - 存取特定受眾的聚合消費數據
        - **總經數據 API:**
          - 依 API 呼叫次數計費, Usage-Based
