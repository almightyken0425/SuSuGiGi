# CLAUDE.md

此目錄依平台拆分，請只讀取需要的文件，不要一次讀入全部。

入口：先讀 `structure.md` 確認目標模組所屬平台，再讀對應的 platform 文件。

| 文件 | 內容 |
|------|------|
| structure.md | 完整 tree 結構 + 模組索引，任何任務的入口 |
| app/index.md | App 7 個 module 摘要索引，做 App 任務的入口 |
| app/\<module\>.md | 對應 module 的完整描述，只讀需要的 module |
| web.md | Web Console modules |
| firebase.md | Firebase modules |
| cloud.md | Cloud Service modules |
| external.md | 外部服務 |

禁止一次讀入 `app/` 目錄下全部文件。
