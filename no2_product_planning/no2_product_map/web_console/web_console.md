# WebConsole — Web 主控台

- **功能：**
    - 桌面版財務資料進階查詢
    - 自訂報表建構與匯出
- **目的：**
    - 延伸行動 App 的財務資料至大螢幕，服務需要進階分析的使用者
- **做法：**
    - Web 前端讀取使用者的雲端同步資料；視圖設定可儲存供重複使用
- **排除：**
    - 即時協作編輯
    - 行動端操作功能
- **利弊：**
    - Web 版觸及桌面用戶，但維護 Mobile + Web 雙平台長期增加工程成本
    - 依賴雲端同步資料，離線或未付費用戶無法使用

---

## QueryEngine — 查詢引擎

- **功能：**
    - JQL 查詢介面，支援條件組合篩選
    - 視圖儲存，可將常用查詢設定為快捷視圖
- **目的：**
    - 讓進階用戶能以精確條件快速定位交易資料
- **做法：**
    - JQL 解析引擎，語法類似 Jira Query Language
    - 視圖設定本地儲存
- **排除：**
    - 程式語言級別的資料查詢，例如 SQL
- **利弊：**
    - JQL 比純 UI 過濾更靈活，但需要使用者學習語法

---

### JQLInterface — JQL 查詢介面

- **功能：**
    - 文字輸入式 JQL 查詢，支援金額、日期、類別、帳戶等條件組合
- **目的：**
    - 提供高靈活度的資料篩選能力，超越固定 UI 過濾的表達極限
- **做法：**
    - JQL 語法解析，轉換為資料庫查詢條件後執行
- **排除：**
    - 語音輸入查詢
- **利弊：**
    - 靈活性高，但對非技術用戶有學習門檻

---

### SavedViews — 視圖儲存

- **功能：**
    - 將常用 JQL 查詢設定儲存為命名視圖
    - 快速切換已儲存的視圖
- **目的：**
    - 讓使用者免於重複輸入常用查詢條件
- **做法：**
    - 視圖設定本地儲存，可自訂命名
- **排除：**
    - 視圖跨用戶共享
- **利弊：**
    - 提升重複查詢效率，但視圖數量增多後需要管理機制

---

## ReportBuilder — 報表建構

- **功能：**
    - 自訂維度報表
    - 進階匯出，支援 CSV、PDF 等格式
- **目的：**
    - 讓進階用戶能依需求建構個人化財務報表
- **做法：**
    - 使用者選定時間、類別、帳戶等維度後產生報表；匯出格式多樣
- **排除：**
    - 即時協作報表
    - 自動定期傳送報表
- **利弊：**
    - 自訂維度靈活，但設定步驟較多，輕度用戶可能少用

---

### CustomDimensions — 自訂維度

- **功能：**
    - 依時間、類別、帳戶等維度組合建構報表
- **目的：**
    - 讓使用者依個人分析需求切分財務資料
- **做法：**
    - 維度選擇介面，選定後即時計算報表數據
- **排除：**
    - 系統外資料的整合維度，例如市場指數
- **利弊：**
    - 高度靈活，但缺乏預設模板時用戶需花時間設定

---

### ExportFunction — 進階匯出

- **功能：**
    - 將報表或交易清單匯出為 CSV 或其他格式
- **目的：**
    - 讓使用者可將資料帶出至外部工具進行進一步分析
- **做法：**
    - 前端直接產生匯出檔案，不依賴後端服務
- **排除：**
    - 自動排程匯出
- **利弊：**
    - 手動匯出簡單易用，但需用戶主動操作，缺乏自動化
